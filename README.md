# Algorithm Visualizer

Interactive visual explanations of core computer science algorithms, built with React, TypeScript, and Vite.

## Overview

`Algorithm Visualizer` is a small educational web app focused on making algorithm behavior intuitive.
Instead of reading pseudocode and mentally simulating each step, users can watch the state change in real time.

The current version includes a sorting playground with:

- Bubble sort visualization
- Insertion sort visualization
- Selection sort visualization
- Merge sort visualization
- Quick sort visualization
- Heap sort visualization
- Adjustable array size
- Adjustable playback speed
- Step-by-step playback controls
- Randomized input generation

## Why I built it

I wanted a project that demonstrates both front-end engineering and core CS fundamentals.
This app combines UI design, state management, animation timing, and algorithm implementation in one place.
